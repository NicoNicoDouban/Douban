# login App
## templates
- signin.html：用户登陆时的页面。  
- signin2.html：用户注册时的页面。  
- signin3.html：用户忘记密码重设时的页面。  
## emailVerify.py
- def   createCode():用于生成邮箱发送时的随机激活码和忘记密码时发送的验证码。  
- def saveCode(user, status):用于保存激活码于userActive表中。  
- def  send_email(username, code, mail):用于发送注册邮件。  
## userForm.py
- class RegistFormR(forms.Form): 用于生成注册时的表单。  
- class RegistFormS(forms.Form)：用于生成注册页面发送验证码的表单。  
- class ForgetFormS(forms.Form)：用于生成忘记密码提交修改时的表单。  
- class ForgetFormR(forms.Form)：用于生成忘记密码页面发送验证码的表单。
## views.py
- def ajax_val(request)：用于动态刷新验证码。  
- def userRegister(request)：用于用户注册的后台代码。  
- def userLogin(request)：用于用户登陆的后台代码。  
- def forget_pwd(request)：用于用户忘记密码的后台代码。  
# Users App
## models.py
- class Users：用于生成用户的个人信息表，包括用户名、邮箱、密码、生日、性别、关注数、注册时间、地址、头像和个性签名等等。  
- class Articles：用于生成文章表，包括标题、作者、发布时间、点击数、内容和点赞数等等。  
- class Books：用于生成书表，包括书名、作者、作者简介、出版社、点赞数、点击数、内容、封面路径等等。  
- class Comments：用于生成书评表，包括评论者、书、评论时间和内容等等。  
- class GoodLink：用于生成点赞关系表，包括点赞者，书和时间等等。  
- class FollowLink：用于生成关注关系表，包括关注者和被关注者等等。  
- class useractive：用于生成激活码表，包括用户的email，激活码和用途等等。  
- class UserCollectionBooks：用于生成用户收藏书籍表，包括用户和书。  
- class UserCollectionArticles：用于生成用户收藏文章表，包括用户和文章。  
## admin.py
- class UserFilterBirthday：生日的过滤器。  
- class UserFilterPubtime：注册时间的过滤器。  
- class CommentFilterPubtime：评论发布时间的过滤器。  
- class ArticleFilterPubtime：文章发布时间的过滤器。  
- class ClickNumFilter：点击数的过滤器。  
- class LickNumFilter：点赞数的过滤器。  
- class UserAdmin：Users表的后台显示。  
- class ArticleAdmin：Aritcles表的后台显示。  
- class BookAdmin：Books表的后台显示。  
- class CommentsAdmin：Comments表的后台显示。  
- class GoodLinkAdmin：GoodLink表的后台显示。  
- class FollowLinkAdmin：FollowLink表的后台显示。  
- class CollectionBooksAdmin：UserCollectionBooks表的后台显示。  
- class CollectionArticlesAdmin：UserCollectionArticles表的后台显示。
